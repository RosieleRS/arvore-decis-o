# 🌳 Árvore de Decisão

Este projeto implementa uma lógica baseada em árvore de decisão para simular a tomada de decisões a partir de condições.

---

## 💡 Descrição

A aplicação utiliza estruturas condicionais para representar um fluxo de decisões, onde cada escolha leva a um novo caminho, simulando um processo lógico semelhante a sistemas inteligentes.

---

## ⚙️ Funcionalidades

* Estrutura de decisões com base em condições
* Simulação de diferentes cenários
* Organização do fluxo lógico de forma sequencial

---

## 🧠 Conceitos aplicados

* Estruturas condicionais (if/else)
* Lógica de programação
* Organização de fluxo de decisão

---

## 🛠️ Tecnologias utilizadas

* Java / JavaScript

---

## 🚀 Como executar

1. Clone o repositório ou baixe os arquivos
2. Execute o projeto no ambiente de desenvolvimento
3. Siga as instruções exibidas no programa

---

## 📌 Objetivo do projeto

Praticar lógica de programação e desenvolver a capacidade de estruturar decisões de forma organizada, simulando cenários do mundo real.
